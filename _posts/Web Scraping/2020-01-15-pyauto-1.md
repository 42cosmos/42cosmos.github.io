### py auto gui

as p

*설정 - 보안 및 개인 정보 보호 - 개인정보보호 - 손쉬운 사용 - 터미널 체크*

마우스와 키보드를 제어한다. > 자동화 프로그램 ( 매크로 ) 생성 시 이용

마우스 좌표를 먼저 알아야 함 : 해상도에 따른 위치 

```python
p.position()
# (x, y)
p.moveTo(x, y(, time))
# 절대적인 위치 변경
p.moveRel(x, y(, time))
# 상대적인 위치 변경
```



#### Click

```python
p.click() # 한 번
p.click(clicks=2, interval=2) # 두 번 클릭, 클릭 사이에 2초 텀
p.doubleClick()
```
