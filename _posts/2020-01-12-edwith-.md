```python
t = np.array([1,2,3,4], float)

t2 = np.array([[1,2,3,4]], float)

# Expected Result - Matrix

```

- shape : numpy array의 object의 dimension 구성을 반환함 > 튜플타입 반환

- dtype : numpy array의 데이터 type을 반환함

```python
np.array(test_matrix).reshape(2,2,2)
```
